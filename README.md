## 🚀 快速开始

1. **注册账号** — 访问 [api.tsbys.com](https://api.tsbys.com) 点击注册
2. **获取 Key** — 登录后台，创建 API Key
3. **配置客户端** — 设置 API Base URL 为 `https://api.tsbys.com`
4. **开始使用** — 选择模型，开始调用

### 示例：OpenAI SDK（Python）

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-你的Key",
    base_url="https://api.tsbys.com/v1"
)

response = client.chat.completions.create(
    model="claude-sonnet-4-6",
    messages=[{"role": "user", "content": "你好！"}]
)
print(response.choices[0].message.content)
```

### 示例：cURL

```bash
curl https://api.tsbys.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```
